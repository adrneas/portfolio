## Contexto

Este case demonstra como uma identidade pode nascer de um princípio simples: a cidade como camada, rota e encontro. O sistema usa blocos, linhas e marcadores para criar peças que funcionam tanto como cartaz quanto como orientação.

## Processo

A exploração começou por mapas, grades de jornal e registros de sinalização temporária. A partir disso, o projeto definiu uma família de layouts que suporta informação densa sem perder ritmo visual.

## Resultado

O resultado é uma linguagem expansível para campanha, site, mapas impressos e chamadas de programação. A identidade evita depender de uma única imagem-chave e funciona como um conjunto de regras.
